Specify a storage account name. Leave blank to create a new storage account.
If using exist storage account name, please use "Use existing" to void typo and replication strategy errors.

The storage account name has to be unique across Azure!
The field can contain only lowercase letters and numbers. Name must be between 3 and 24 character. Attempting to use other characters, starting the Name with a non-letter character, or entering a name that is identical to that of another extended property owned by the same storage account, will result in a status code 400 (Bad Request) error.
